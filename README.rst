Oriole-Service
==============

|Eric's world| |Eric's world|

**Rapidly create services.**

软件平台的搭建框架，更快，更好，更高。

Prerequisites 系统软件
----------------------

-  python3
-  mongodb
-  mysql
-  rabbitmq
-  redis

Install 安装
------------

::

      pip install oriole-service
